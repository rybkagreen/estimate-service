## Компоненты Estimates Page

### 1. Навигация
- Кнопки для работы с различными статусами смет (Все, В работе, Завершенные, Черновики)
- Кнопка для создания новой сметы

### 2. Фильтры
- Контекстные фильтры для быстрого поиска смет
- Многоуровневые опции фильтрации

### 3. Список смет
- Отображение ключевой информации по каждой смете
- Сортировка и поиск по различным параметрам

### 4. Детали сметы
- Полная информация по выбранной смете
- Возможность редактирования и экспорта
- Список позиций с возможностью редактирования

### 5. Взаимодействие
- Контекстные действия для работы с выбранной сметой
- Опции импорта/экспорта сметных данных
